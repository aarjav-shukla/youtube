📺 YouTube Clone — Static
A pixel-faithful static clone of the YouTube homepage built with pure HTML and CSS, replicating the core layout and UI components of YouTube's interface.
🔗 Live Demo: aarjav-shukla.github.io/youtube-clone-static

📸 Preview

A static recreation of the YouTube homepage featuring the header, sidebar, and video grid layout.


✨ Features

Header — YouTube logo, search bar with voice search button, notification bell, and user profile icon
Sidebar — Home, Shorts, Subscriptions, You, Explore, and More from YouTube sections with icons
Video Grid — Responsive grid of video cards with thumbnails, video duration overlay, channel icons, titles, view counts, and upload dates
Google Fonts — Uses the Roboto font family, matching YouTube's typography


🗂️ Project Structure
youtube-clone-static/
│
├── index.html              # Main HTML file
├── style.css               # All styles
├── temp.js                 # JavaScript (minor interactions)
│
├── icons/                  # UI icons (hamburger menu, search, voice, notifications, etc.)
├── thumbnails/             # Video thumbnail images
├── channel thumbnails/     # Channel profile pictures
├── sidebar logos/          # Primary sidebar icons and subscription avatars
└── sidebar2logos/          # Secondary sidebar icons (Shorts, History, Playlists, etc.)

🚀 Getting Started
No installation or build step required — it's fully static.

Clone the repository

bash   git clone https://github.com/aarjav-shukla/youtube-clone-static.git

Open in browser

bash   cd youtube-clone-static
   open index.html
Or simply double-click index.html to open it in your default browser.

📌 Notes

This is a static UI clone — no backend, no real video playback, and no YouTube API integration.
Built purely for front-end practice and learning purposes.
All thumbnails and channel images are local assets used for demonstration only.
